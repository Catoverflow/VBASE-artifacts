template<typename T, DataType Dt>
class milvus::FieldData< T, Dt >

Template class represents column-based data of a field. Avaiable inheritance classes:
BoolFieldData for boolean scalar field
Int8FieldData for 8-bits integer scalar field
Int16FieldData for 16-bits integer scalar field
Int32FieldData for 32-bits integer scalar field
Int64FieldData for 64-bits integer scalar field
FloatFieldData for float scalar field
DoubleFieldData for double scalar field
StringFieldData for string scalar field (not supported in 2.0, reserved)
BinaryVecFieldData for float vector scalar field
FloatVecFieldData for binary vector scalar field
